upinside 는 디시인사이드 갤러리 속 게시글을 외부 데이터베이스에 미러링하기 위한 어플리케이션 입니다.

## 설치 및 실행
> 다음과 같은 추가 프로그램이 필요합니다.
* NodeJS >= 10.x
* MongoDB >= 4.x

```
$ npm install
```

* 장시간 실행시 발생하는 오류 없는지 확인
* 설정 (PM2)
* 디스코드 연동
* 웹 프론트엔드
